# Подсистема СКУД

## Dev режим
_запуск сервера_

`make start-dev`

_остановка сервера_

`make stop-dev`

## Prod режим
_запуск сервера_

`make start-prod`

_остановка сервера_

`make stop-prod`

## Работа с БД
_обновление базы_

`make update-db`

_удаление базы_

`make drop-db`

## Env переменные

Для отправки отчетов на почту требуется прописать env переменные:

`MAIL_USERNAME=ХХХХХХ`

`MAIL_PASSWORD=ХХХХХХ`
